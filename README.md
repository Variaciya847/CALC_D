# Калькулятор на C++ с поддержкой различных операций

Проект представляет собой расширенный калькулятор на C++ с поддержкой различных математических и бухгалтерских операций, логированием и историей операций.

## 📋 Функциональность

### Основные модули:
1. **Арифметические операции** - сложение, вычитание, умножение, деление
2. **Тригонометрические функции** - sin, cos, tg, ctg с вычислением через ряд Тейлора
3. **Логарифмы** - натуральный, десятичный, логарифм по произвольному основанию
4. **Логические операции** - побитовые И, ИЛИ, XOR, НЕ
5. **Бухгалтерский баланс** - учет проводок с проверкой баланса
6. **Конвертация единиц** - метрические и имперские единицы длины
7. **Работа с датами** - операции с датами, определение дней в месяце
8. **Строковые операции** - парсинг и вычисление математических выражений из строк
9. **История операций** - отмена/повтор операций, просмотр истории

Установка и сборка

### Предварительные требования
- CMake 3.15+
- Компилятор с поддержкой C++20
- spdlog (автоматически загружается через CMake)

### Сборка проекта:
```bash
# Клонирование репозитория
git clone <repository-url>
cd calculator

# Создание директории для сборки
mkdir build && cd build

# Конфигурация и сборка
cmake ..
cmake --build .

# Запуск программы
./calculator
